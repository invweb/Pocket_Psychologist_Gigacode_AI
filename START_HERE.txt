═══════════════════════════════════════════════════════════════════════════════
                         БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════════════════════

1. Откройте проект в Android Studio (Hedgehog или новее)
2. Дождитесь завершения синхронизации Gradle (внизу экрана)
3. Подключите Android устройство или запустите эмулятор
4. Нажмите кнопку Run (зеленый треугольник в верхней панели)

Приложение будет установлено и запущено на устройстве.

═══════════════════════════════════════════════════════════════════════════════
                         ФУНКЦИОНАЛ
═══════════════════════════════════════════════════════════════════════════════

Анализ голоса:
  1. Нажмите "Начать запись"
  2. Говорите 10-30 секунд
  3. Подождите завершения анализа
  4. Увидите настроение в виджете

Дыхательные упражнения:
  1. Нажмите "Сделайте глубокий вдох"
  2. Следуйте инструкциям (вдох-задержка-выдох-задержка)
  3. Завершите упражнение

Виджет:
  1. Проведите анализ
  2. Нажмите и удерживайте на главном экране
  3. Выберите "Настроение дня"

                         ПРОБЛЕМЫ И РЕШЕНИЯ
═══════════════════════════════════════════════════════════════════════════════

Проблема: "Unable to locate a Java Runtime"
Решение: Установите JDK 11 или новее
         export JAVA_HOME=$(/usr/libexec/java_home -v 11)

Проблема: Gradle синхронизация не завершается
Решение: ./gradlew clean && ./gradlew build --refresh-dependencies

Проблема: Устройство не определяется
Решение: adb kill-server && adb start-server && adb devices
