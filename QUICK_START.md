# Карманный психолог - Инструкция по запуску

## Быстрый старт

1. Откройте проект в Android Studio (Hedgehog или новее)
2. Дождитесь завершения синхронизации Gradle
3. Подключите Android устройство или запустите эмулятор
4. Нажмите кнопку Run (зеленый треугольник)

## Возможные проблемы и решения

### Проблема: "Unable to locate a Java Runtime"

**Решение:** Установите JDK 11 или новее:
```bash
# Проверьте установленные версии Java
/usr/libexec/java_home -V

# Если Java не установлена, установите через Homebrew
brew install openjdk@11

# Установите JAVA_HOME
export JAVA_HOME=$(/usr/libexec/java_home -v 11)
export PATH=$JAVA_HOME/bin:$PATH
```

### Проблема: Gradle синхронизация не завершается

**Решение:** Очистите кэш и пересинхронизируйте:
```bash
cd /Users/vasiliikarpenko/AndroidStudioProjects/PocketPsychologist
./gradlew clean
./gradlew --stop
./gradlew build --refresh-dependencies
```

### Проблема: Устройство не определяется

**Решение:** Перезапустите ADB:
```bash
adb kill-server
adb start-server
adb devices
```

## Использование приложения

### Анализ настроения

1. Откройте приложение "Карманный психолог"
2. Нажмите кнопку **"Начать запись"** (синяя кнопка с иконкой микрофона)
3. Проговорите что-нибудь в течение 30 секунд
4. Подождите завершения анализа (появится прогресс-бар)
5. Увидите свое настроение в виджете "Настроение дня" на главном экране

### Дыхательные упражнения

1. Нажмите на карточку **"Сделайте глубокий вдох"**
2. Следуйте инструкциям на экране:
   - **Вдох** (зеленый круг увеличивается)
   - **Задержка** (желтый круг)
   - **Выдох** (зеленый круг уменьшается)
   - **Задержка** (желтый круг)
3. Повторяйте упражнение 3-5 минут для расслабления

### Виджет настроения

1. Запустите приложение и проведите анализ голоса
2. Нажмите и удерживайте на главном экране
3. Выберите "Настроение дня"
4. Разместите виджет на главном экране для быстрого доступа

## Разрешения

При первом запуске приложение запросит:
- **Запись аудио** - для анализа голоса
- **Вибрация** - для ритмичных упражнений
- **Уведомления** - для фонового сервиса записи

## История настроений

- История автоматически сохраняется в локальной базе данных
- Последние 5 записей отображаются на главном экране
- Каждая запись содержит: тип настроения, скорость речи, уверенность анализа

## Поддержка

При возникновении проблем:
1. Проверьте логи: `adb logcat | grep "PocketPsychologist"`
2. Убедитесь, что микрофон не занят другими приложениями
3. Проверьте, что на устройстве достаточно свободного места
